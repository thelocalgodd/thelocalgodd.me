*a story of how I got into coding, as a fresher in KNUST.*

## first steps — late first semester, 2023

my journey began unexpectedly. I had started a four-year Computer Science program at KNUST, although I was expecting to pursue Electrical Engineering. while things didn't go that way (perhaps for the best!), I found myself in Introduction to Structured Program Design during my first semester.

knowing Introduction to Programming in C++ was coming in the second semester, I decided to get a head start. I got onto w3schools and learned the basics of C++. it was bliss — I found it very interesting and fun.

to keep my skills sharp, I started tackling a set of 49 practice questions by Dr. Hayfron Acquah, which were also potential exam questions. I solved what I could and uploaded them to GitHub.[^1] the remaining unsolved questions are still out there; maybe I'll revisit them.

when the actual C++ course started, I found it easy and did well in the end.

## the long vacation — august 2023 to january 2024

exams finished and it was time for the long vacation — four glorious months. with so much time and endless possibilities, I planned to work on projects with friends. some thrived, some fizzled out, and some never even started, but they all provided valuable experience.

to maximize my learning, I explored a different technology each month.

**Month 1** — HTML, CSS, and Linux. I grasped these concepts well.

**Month 2** — JavaScript (a 30-day course that was great, though some parts of JS were... well, challenging).

**Month 3** — planned to learn React but got sidetracked. still gained a basic understanding, which proved helpful later.

**Month 4** — project building month. I focused on creating projects, most of which are now on my GitHub profile. one of them is my first personal portfolio website.

the long vacation was a worthwhile investment. I picked up valuable skills and made online friends along the way.

## present

back in school, I came across a few GitHub repositories and discovered Astro — the framework I originally used to build this site.
